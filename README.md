# PARCOURINF

Simulateur intelligent d'accessibilité aux formations après le bac.

##  Tester le site

-> https://parcourinf.onrender.com

##  Description

PARCOURINF est une application web permettant de simuler un parcours d'admission avec :
- une API FastAPI
- un modèle de Machine Learning
- une simulation de la phase principale
- une interface utilisateur
  c'est une équivalence de Parcoursup, en simulation

##  Technologies utilisées

- Python
- FastAPI
- HTML/CSS
- Scikit-learn
- Render
